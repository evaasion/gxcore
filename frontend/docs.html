CypherSolBase Documentation

Complete guide to integrating CypherSolBase in your Solana projects

Installation

Add CypherSolBase to your Cargo.toml:

[dependencies]
cyphersolbase = "0.1.0"

Or install from GitHub:

cargo add --git https://github.com/evaasion/gxcore.git cyphersolbase

Basic Usage

Encoding Data

use cyphersolbase::{encode, CompressionAlgorithm};

let data = b"Hello, Solana!".to_vec();
let seed = b"my_secret_seed".to_vec();
let encoded = encode(&data, &seed, CompressionAlgorithm::Lz4)?;

Decoding Data

use cyphersolbase::decode;

let decoded = decode(&encoded, &seed)?;

Verifying Integrity

use cyphersolbase::partial_verify;

let is_valid = partial_verify(&encoded)?;

API Reference

Functions

  • encode(data: &[u8], seed: &[u8], compression: CompressionAlgorithm) -> Result> - Encode data with dynamic alphabet
  • decode(encoded: &[u8], seed: &[u8]) -> Result> - Decode data
  • partial_verify(encoded: &[u8]) -> Result - Verify CRC-32 without decoding

Enums

  • CompressionAlgorithm { None, Lz4, Brotli } - Compression options

Web API

CypherSolBase provides a REST API for testing and integration:

Endpoints

  • POST /encode - Encode data. Body: { "data": [u8], "seed": [u8], "compression": "none|lz4|brotli" }
  • POST /decode - Decode data. Body: { "encoded": [u8], "seed": [u8] }
  • POST /verify - Verify data. Body: { "data": [u8], "seed": [u8], "encoded": [u8] }

Live API: https://gxcore.onrender.com

Examples

Solana Program Integration

use cyphersolbase::{encode, CompressionAlgorithm};
use solana_program::msg;

// In your Solana program
let payload = encode(&user_data, &program_seed, CompressionAlgorithm::Lz4)?;
msg!("Encoded payload: {:?}", payload);

DeFi Transaction Security

// Secure sensitive transaction data
let secure_data = encode(&transaction_details, &user_seed, CompressionAlgorithm::None)?;
let verified = partial_verify(&secure_data)?;
