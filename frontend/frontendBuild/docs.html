git clone https://github.com/evaasion/gxcore.git cd gxcore cargo build --release

Quick Tip

cyphersolbase = "=0.1.0"

API Reference

cyphersolbase::encode(data: &[u8], seed: &[u8], compression: Compression) -> Result

Encodes binary data using dynamic Base64 alphabet with optional compression and privacy layers.

Parameters:

  • data: Binary data to encode
  • seed: Seed for dynamic alphabet generation
  • compression: Compression algorithm (None, LZ4, Brotli)

Returns:

Base64-encoded string with integrity checksum

cyphersolbase::decode(encoded: &str, seed: &[u8], compression: Compression) -> Result>

Decodes CypherSolBase encoded string back to original binary data.

Parameters:

  • encoded: CypherSolBase encoded string
  • seed: Same seed used for encoding
  • compression: Same compression used for encoding

Returns:

Original binary data

cyphersolbase::verify(encoded: &str) -> Result

Verifies the integrity of encoded data using CRC-32 checksum.

Parameters:

  • encoded: CypherSolBase encoded string

Returns:

True if data integrity is verified, false otherwise

Usage Examples

Basic Encoding

use cyphersolbase::{encode, Compression};

let data = b"Hello, Solana!";
let seed = b"my-secret-seed";

let encoded = encode(data, seed, Compression::None)?;
println!("Encoded: {}", encoded);

With Compression

use cyphersolbase::{encode, decode, Compression};

let data = b"Large dataset that benefits from compression...";
let seed = b"compression-seed";

let encoded = encode(data, seed, Compression::LZ4)?;
let decoded = decode(&encoded, seed, Compression::LZ4)?;

assert_eq!(data.to_vec(), decoded);
